== Installation ==

= Instalação =
* Upload to /wp-content/plugins/directory
* Check if you have already installed the Plugins is essential for the functioning of the plugin.
* Activate the plugin via the 'Plugins' menu in WordPress.
* Go to the > Form Quiz menu.
